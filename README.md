# MobileHub Pro 📱

MobileHub Pro is a modern, fully responsive e-commerce web application designed for browsing and purchasing mobile smartphones. It features a clean user interface (UI) and an intuitive user experience (UX).

---

## 🚀 Features

* **Fully Responsive Design:** Optimized for a seamless viewing experience across desktops, tablets, and mobile devices.
* **Modern & Clean UI:** Aesthetic design with smooth layouts to keep users engaged.
* **Product Showcase:** Comprehensive listing of the latest smartphones (including iPhone, Samsung, OnePlus, Vivo, and Xiaomi).
* **Interactive Shopping Cart:** Allows users to easily add, manage, and view items in their cart.
* **Live Search Bar:** Quick and responsive product filtering to find specific phone models instantly.

---

## 🛠️ Technologies Used

This project is built using standard front-end web technologies:
* **HTML5** - For structuring the web pages and content.
* **CSS3** - For styling, layout design, and responsive media queries.
* **JavaScript (ES6)** - For handling cart functionalities, dynamic interactions, and search logic.
